რა არის Bubbling და Capturing JavaScript-ში Event Flow-ს ორი ძირითადი ფაზა არსებობს: Capturing (დაჭერის ფაზა) იწყება მაღლიდან დაბლა — ანუ document → html → body → parent → child. ბრაუზერი პირველ რიგში ამოწმებს ყველაზე გარე ელემენტს და ნელ-ნელა ჩამოდის შიდა ელემენტამდე. Capturing-ის დროს იშვიათად ვიყენებთ, მაგრამ შეიძლება მოვითხოვოთ {capture: true} პარამეტრით. Bubbling (ამოსვლის ფაზა) იწყება შიდა ელემენტიდან გარეთ — ანუ child → parent → body → html → document. ნაგულისხმევად (default) addEventListener იყენებს Bubbling-ს. ეს უფრო ხშირად გამოიყენება, რადგან event delegation ძირითადად ბაბლინგზე მუშაობს.
